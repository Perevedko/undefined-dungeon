# undefined-dungeon
online multiplayer roguelike game written on vue.js &amp; sinatra.rb

---
### Server requirements 
- Ruby 2.6.3
- Mongodb server  
  by default, application will try to connect to `localhost:27017`  
  you can configure it at `server/config/mongoid.yml`
